# AxVirtIO Devices

A collection of `no_std` compatible VirtIO device implementations for the [ArceOS-Hypervisor](https://github.com/arceos-hypervisor/) project.

## Crates

| Crate | Description |
|-------|-------------|
| **axvirtio-common** | Common types, traits, and utilities for VirtIO devices |
| **axvirtio-blk** | VirtIO block device implementation with MMIO transport |
